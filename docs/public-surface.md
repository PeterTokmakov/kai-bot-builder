# Public surface decision

Chosen public GitHub surface: **public product/demo/docs repo**.

Why this surface:

- honest public artifact for GitHub discovery
- safe to publish without exposing private production internals as if they were a runnable OSS release
- enough for GitHub search/topics, article backlinks, and future comparison pages

Not unlocked yet:

- awesome-selfhosted
- source-first OSS lists
- LibHunt if it expects a true public source/release posture

Those should wait for a genuinely public runnable repo.
